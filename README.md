# Python Basic Projects: Calculator & Games

This repository contains simple yet functional Python projects developed during my initial learning phase of Python programming.

## Projects Included:
1. **Terminal-Based Calculator:** A command-line interface tool that can perform basic arithmetic operations like addition, subtraction, multiplication, and division.
2. **Rock, Paper, Scissors Game:** A classic interactive game played against the computer using Python's `random` module.

## How to Run:
- Ensure you have Python installed.
- Clone the repository.
- Run the files using:
  ```bash
  python calculator.py
  python game.py
